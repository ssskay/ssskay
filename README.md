## ⭐ Stuff I've built

The ones for other people to actually use.

### 🤖 Agentic tools — you talk, they work

Each one ships with a Claude Code skill, and each one has a line the AI can't cross. That's the feature.

🏛️ **[internet-historian](https://github.com/ssskay/internet-historian)** — a patient Wayback Machine archiver for macOS that treats a busy Internet Archive as weather, not failure — it retries forever until your pages are preserved. Say *"archive Chiikawa stuff"* and it handles the rest. `pipx install internet-historian`. *Web pages die. Yours don't have to.*

📖 **[wiki-gap-finder](https://github.com/ssskay/wiki-gap-finder)** — finds people from underrepresented groups who should have an English Wikipedia article but don't, then fact-checks every claim into a research dossier. Zero API keys, and it never writes the article — that's yours. *AI found her and checked the facts; a human wrote her story.*

📦 **[inbox-catalog](https://github.com/ssskay/inbox-catalog)** — local-first, provably read-only engine that turns your own order emails into a catalog of what you own, plus an Amazon returns tracker with a return-window clock. Installs as a Claude Code plugin with a zero-setup demo. *Your inbox already knows everything you've bought.*

### 🎁 Free stuff engines

🌲 **[Free Stuff @ Dartmouth](https://github.com/ssskay/freestuff-dartmouth-v2)** — the Dartmouth reference build. *"Your school already pays for this. You just didn't know."*
